Store Management System
A full-stack web application developed as part of an academic assignment to manage a store's products, categories, customers and orders.

About The Project
This system was built using the Database-First approach with ASP.NET MVC and Entity Framework. It allows store staff to manage all core store operations through a clean web interface, including a live dashboard showing key business metrics.

Features
Dashboard – Live summary of total customers, products, orders, categories and revenue
 Product Management – Add, edit, delete and view products with stock and category tracking
 Category Management – Organise products into categories
 Customer Management – Store and manage customer records
 Order Management – Create and track orders with status and shipping address
